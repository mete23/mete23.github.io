About PhotoTool

PhotoTool is a web application that allows you to capture photos using your webcam and apply various filters and effects to enhance your images.
When capturing a photo there is a crosshair, showing the device orientation. It also provides basic image manipulation functionality.
There is also an Undo and Reset Button to revert changes. But beware, that undo only works for the last change. You can save your edited photos by pressing on the "Save Photo" button.

Features

  • Capture photos using your webcam
  • Apply filters like Grayscale, Sepia, Invert, and more
  • Adjust image properties such as Brightness, Contrast, and Saturation
  • Save and download your edited photos
  • Undo changes and reset to the original photo
  • Request Gyroscope permissions for additional features
  • Responsive design for mobile and desktop devices

Instructions

  1. Navigate to the Camera tab to capture a photo.
  2. Switch to the Editing tab to apply filters and effects.
  3. Use the buttons provided to adjust the image properties.
  4. Click the Save photo button to download your edited image.
  5. Use the Undo button to revert the last change.
  6. Click the Reset button to discard all changes and start over.

Technical Details

The application is built using HTML, CSS, and JavaScript. It leverages the getUserMedia API to access the webcam and capture photos.
The captured image is drawn onto a hidden canvas element, where various filters and effects are applied using CSS and JavaScript.
The edited image can then be saved to the user's device.

Disclaimer

On iPhone, you need to click the "Request Gyro Permissions" button first and then reload the website for the permissions to take effect.

Supported Browsers and Devices

Some of the filters like Blur and Sharpen and Hue Rotation may not work on all browsers.
With Chrome on Windows on a Laptop with a Motion Sensor everything works.
On Mobile phones, the most can be used by using Chrome on an Android device. Only some of the Filters like Blur and Sharpen may not work.
